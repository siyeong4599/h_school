< DOM >

Document Object Model의 약자로 HTML 또는 XML 문서에
접근하기 위해 사용할 함수와 객체의 인터페이스를
정의한 규약

자바스크립트는 위의 DOM을 구현한 객체인 document를 
제공하므로 이를 사용하여 HTML 문서를 조작할 수
있습니다.
s

엘리먼트 노드(element node): 태그 정보가 들어 있는 노드
텍스트 노드(text node): 태그 안의 문자열 정보가 들어 있는 노드
속성 노드( node): 택이 속성 정보가 들어 있는 노드
주석 노드(comment node): 주석 정보가 들어 있는 노드
문서 노드(document node): 문서 전체를 가리키는 노드
